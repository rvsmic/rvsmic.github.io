# Portfolio

This is a source code repo for my portfolio website available at [rvsmic.github.io](https://rvsmic.github.io). Enjoy :)

# Technical Details

## Technologies Used

- Vue.js
- Tailwind CSS

## Usage

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
